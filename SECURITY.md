## 支持的版本

我们当前支持以下版本的安全更新：

| 版本 | 状态 |
|------|------|
| latest | ✅ 支持 |
| main branch | ✅ 支持 |

## 报告安全漏洞

如果您发现了安全漏洞，请通过以下方式报告：

1. **不要** 在公开的 issue 或 PR 中报告安全问题
2. 发送邮件至项目维护者
3. 我们会在 24 小时内确认收到报告
4. 我们会在 7 天内提供初步评估

## 安全最佳实践

### API Key 安全

- **永远不要** 将真实的 API Key 提交到版本控制
- 使用 `.env` 文件存储敏感信息
- 确保 `.env` 已被 `.gitignore` 排除
- 定期轮换 API Key

### 本地运行

- 建议在本地环境运行，避免在公网暴露 API Key
- 使用 `.env` 文件配置敏感信息
- 不要在共享机器上保留 `.env` 文件

### 数据安全

- 训练数据可能包含敏感信息，请妥善保管
- 建议使用本地数据库而非远程数据库
- 定期清理临时文件和缓存

## 依赖安全

本项目依赖以下关键包，请确保及时更新：

- `openai` - OpenAI API 客户端
- `pydantic` - 数据验证
- `sqlalchemy` - 数据库 ORM
- `torch` - PyTorch 深度学习框架

建议定期运行以下命令检查依赖安全：

```bash
pip install safety
safety check
```

## 合规性

- 本工具仅用于合法的模型微调目的
- 请遵守 OpenAI 的使用条款
- 确保您的训练数据来源合法
